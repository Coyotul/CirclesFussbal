\doxysubsection{Detailed Description}
Abstract base class for all command types. 

The \doxylink{class_command}{Command} class is an abstract base class for implementing the \doxylink{class_command}{Command} design pattern. It defines an interface for executing commands. Concrete command classes will derive from this class and implement the {\ttfamily \doxylink{class_command_a6fd7d9bd8df8bfc881e4d6c7cd1878b7}{execute()}} method to perform specific actions.
